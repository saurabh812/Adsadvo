# Adsadvo

## Overview
Adsadvo is a React-based landing page for India's Most Trusted Dropshipping Platform. This is a frontend-only application built with Create React App.

## Project Structure
- `src/` - React source code
  - `components/` - Reusable UI components (Header, Footer, Hero, FAQ, etc.)
  - `pages/` - Page components (Home, About, Pricing, Contact, etc.)
  - `styles/` - Global CSS styles
  - `assets/` - Images and media files
- `public/` - Static public assets

## Tech Stack
- React 18
- React Router DOM 7
- Create React App (react-scripts)

## Development
The app runs on port 5000 using Create React App's development server.
